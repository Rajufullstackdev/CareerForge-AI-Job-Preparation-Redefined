# CareerForge AI – Job Preparation Redefined

## Overview
CareerForge AI is a modern and responsive career preparation website built using HTML5, CSS3, and JavaScript.  
This project helps users explore career guidance, job preparation resources, and AI-inspired features through a clean and interactive user interface.
